## 🧠 What I learned

### 🖥️ What is a Computer System?

A computer system is made up of different hardware components that work together to process data and provide services to users.

---

## 🔌 Key Components of a Computer System

### 🧩 Motherboard
- The backbone of the computer
- Connects all components together
- Contains:
  - CPU socket
  - RAM slots
  - Expansion slots
  - Ports

💡 Acts like the body’s **nervous system**

---

### 🧠 CPU (Central Processing Unit)
- The “brain” of the computer
- Executes instructions and processes data
- Modern CPUs have multiple cores for parallel processing

---

### ⚡ RAM (Random Access Memory)
- Temporary (short-term) memory
- Stores data currently being used by the CPU
- Volatile → data is lost when power is off

💡 Faster access = better performance

---

### 💾 Storage (SSD / HDD)
- Long-term memory of the system

#### HDD:
- Uses moving parts
- Slower but cheaper
- Larger storage capacity

#### SSD:
- No moving parts
- Much faster
- More reliable

---

### 🌐 Network Adapter
- Allows the computer to communicate with other systems
- Can be:
  - Wired (Ethernet)
  - Wireless (Wi-Fi)

---

### 🔋 Power Supply Unit (PSU)
- Provides power to all components
- Converts electricity from outlet into usable power

⚠️ If insufficient → system failure

---

### 🎮 Graphics Card (GPU)
- Processes visual data
- Sends output to monitor

💡 Important for:
- Gaming
- Video editing
- Graphics-heavy tasks

---

### ⌨️ Input / Output Devices

#### Input Devices:
- Keyboard
- Mouse
- Microphone
- Scanner

#### Output Devices:
- Monitor
- Printer
- Speakers

Common connections:
- USB
- HDMI
- DisplayPort

---

## 📌 Key Concept

- Each component has a **specific role**
- All components work together through the **motherboard**
- Efficient communication between components is essential for system performance
